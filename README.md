# 볼러 지원 커뮤니티 - 볼치자

## 🚀기능 목록

- 에버리지 결산 그래프
    - 사용자의 게임 총 점을 입력 받은 일자를 기준으로 분류
    - 일자 별로 저장 후 각 일자의 평균 점수를 저장
    - 일자별 평균점수를 그래프에 출력
    - 일자 별 high 점수와 기존에 저장된 totalHigh점수를 비교하여 최고점 업데이트 및 출력
- 볼링 용품 스토어 모음
    - 온라인 프로샾, 볼링용품 팜매점 등의 링크를 모아놓은 페이지
    - 하이퍼링크를 통해 이미지를 클릭하면 웹페이지로 이동
- 볼링장 지도
    - 사용자의 현재 위치를 기반으로 주변 볼링장 위치를 지도에 마킹해주는 페이지
